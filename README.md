
 🧮 ArthAI – Smart Income Tax Assistant

ArthAI  is an AI-powered smart income tax assistant designed to simplify Indian Income Tax calculation, understanding, and decision-making.  
The platform focuses on tax awareness, tax regime comparison, and AI-based explanations, rather than just form filling.

 💡 Solution – ArthAI

ArthAI provides:

- Accurate rule-based income tax calculation
- Old vs New Tax Regime comparison
- AI-powered explanations using Gemini AI
- Smart tax-saving insights
- Interactive and user-friendly interface


Installation Steps
Clone repository
    git clone https://github.com/Sayali-Yadav16/arth_ai.git
    cd arth_ai
    
Install Dependencies
      npm install or yarn install
      Start the development server
      npm run dev or yarn dev
Usage
    Open the application in your browser.
